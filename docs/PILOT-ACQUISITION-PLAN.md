## 2. Profil Cible

### Fromagerie Ideale

| Critere | Detail |
|---------|--------|
| **Taille** | 5 a 50 employes |
| **Chiffre d'affaires** | 10M a 100M DZD/an |
| **Type de production** | Fromage frais, camembert, pate molle, pate pressee, mozzarella, ricotta |
| **Localisation prioritaire** | Alger, Blida, Tizi-Ouzou, Bejaia, Boumerdes, Medea |
| **Gestion actuelle** | Papier, cahiers, Excel, ou WhatsApp pour les commandes |
| **Decision** | Gerant (proprietaire) ou Directeur de Production |
| **Maturite digitale** | Possede un smartphone, utilise WhatsApp, a eventuellement une page Facebook |

### Points de Douleur Cles

1. **Stock en aveugle** : pas de visibilite sur les matieres premieres (lait, presure, ferments), ruptures frequentes
2. **Tracabilite inexistante** : impossible de remonter un lot en cas de probleme sanitaire
3. **Pertes sur peremption** : fromages frais perdus faute de suivi FIFO
4. **Litiges livraison** : clients contestent les quantites, pas de bon de livraison numerique
5. **Cout de revient flou** : le gerant ne sait pas combien lui coute reellement un kg de fromage
6. **Stress reglementaire** : controles CACQE/DSA de plus en plus frequents, besoin de registres propres

### Red Flags (Ne Pas Cibler)

- Plus de 100 employes (trop complexe, cycle de vente long, besoin SAP/Sage)
- Moins de 5 employes (pas de budget, adoption difficile)
- Deja equipe d'un ERP (Sage, Odoo, ou equivalent)
- Pas de connexion Internet stable dans l'usine
- Gerant non joignable ou delegue tout a un comptable externe

## 5. Script Demo -- 15 Minutes

### Phase 1 -- Validation du Probleme (2 min)

```
Objectif : Faire parler le gerant de ses douleurs actuelles.
```

**Vous** : "Merci de m'accorder ce temps, [Prenom]. Avant de vous montrer quoi que ce soit, j'aimerais comprendre comment vous fonctionnez aujourd'hui. Comment est-ce que vous gerez votre stock de matieres premieres -- le lait, la presure, les ferments ?"

*Ecouter. Prendre des notes. Relancer si besoin :*

- "Et pour la tracabilite des lots, comment vous faites si un client signale un probleme ?"
- "Les livraisons, c'est sur bon papier ? Il y a parfois des contestations ?"
- "Vous savez combien vous coute exactement un kilo de [son produit principal] ?"

**Transition** : "D'accord, c'est exactement pour ces problemes-la qu'on a cree Manchengo. Laissez-moi vous montrer."

### Phase 2 -- Demo Live (10 min)

```
Objectif : Montrer les 4 ecrans cles en conditions reelles.
```

**Ecran 1 -- Tableau de Bord (1 min)**
"Voici votre tableau de bord. En un coup d'oeil, vous voyez votre stock, vos productions du jour, votre chiffre d'affaires. Tout est en temps reel."

**Ecran 2 -- Gestion du Stock (3 min)**
"Ici, votre stock de matieres premieres. Chaque reception de lait est enregistree avec le fournisseur, la quantite, le lot. Le systeme vous alerte automatiquement quand un produit est proche de la peremption ou quand le stock est bas."

*Montrer :*
- Ajout d'une reception de lait
- Alerte de stock bas
- Vue des lots avec dates de peremption

**Ecran 3 -- Production (3 min)**
"Quand vous lancez une production, vous selectionnez la recette -- par exemple, votre camembert. Le systeme calcule automatiquement les matieres premieres necessaires et les deduit du stock. Chaque lot produit est trace."

*Montrer :*
- Lancement d'un ordre de production
- Deduction automatique du stock
- Numero de lot genere

**Ecran 4 -- Livraisons et Facturation (3 min)**
"Pour les livraisons, vous creez un bon numerique. Le client peut meme signer sur l'ecran. Plus de contestation. Et la facture se genere automatiquement."

*Montrer :*
- Creation d'un bon de livraison
- Facture automatique
- Historique client

### Phase 3 -- Cloture (3 min)

```
Objectif : Proposer le pilote et obtenir un engagement.
```

**Vous** : "Voila en gros comment ca marche. Est-ce que ca repond aux problemes dont on a parle au debut ?"

*Ecouter la reaction.*

**Vous** : "On peut vous installer tout ca gratuitement pendant 3 mois. On s'occupe de tout : la creation de votre compte, l'import de vos produits, et on vous forme en une heure. Tout ce qu'on demande, c'est votre avis toutes les deux semaines pour qu'on ameliore le produit."

*Si hesitation :*
- "Il n'y a aucun engagement. Apres les 3 mois, vous decidez si vous continuez ou pas."
- "Vos donnees vous appartiennent a 100%. Si vous arretez, on vous les exporte."
- "On n'a que 5 places sur ce programme pilote, et on en a deja [X] de prises."

**Si OK** : "Super ! Je vous envoie un petit document par WhatsApp, c'est une convention simple d'une page. Et on fixe la date d'installation ensemble."

**Si pas maintenant** : "Pas de probleme. Je vous envoie un recapitulatif par WhatsApp et on en reparle quand vous etes pret. Est-ce que la semaine prochaine ca irait pour revenir vers vous ?"

**Etape 1 -- Configuration du compte (15 min)**

- [ ] Parametrage du profil entreprise (nom, adresse, logo, RC)

**Etape 2 -- Import du catalogue produits (15 min)**
- [ ] Import des matieres premieres (lait, presure, ferments, emballages)

**Etape 4 -- Creation des comptes utilisateurs (10 min)**

- [ ] Compte pour le responsable production (role : Production)
- [ ] Compte pour le magasinier/stockiste (role : Stock)
- [ ] Compte pour le commercial/livreur (role : Ventes)

**Etape 5 -- Formation pratique (20 min)**
- [ ] Simuler une reception de matiere premiere
- [ ] Simuler un lancement de production
- [ ] Simuler une livraison avec bon et facture
- [ ] Montrer le tableau de bord et les alertes
- [ ] Repondre aux questions

- [ ] Creer un groupe WhatsApp dedie : "Manchengo x [Nom fromagerie]"

- [ ] Verifier les premieres saisies dans le systeme (donnees coherentes ?)

## Rappels Cles

1. **WhatsApp est roi** : En Algerie, le business se fait sur WhatsApp. L'email est secondaire.
2. **Le gerant decide seul** : Pas de comite d'achat. Convaincre une personne suffit.
3. **La demo en personne vaut 10x la visio** : Si possible, se deplacer.
4. **Le gratuit rassure mais inquiete** : Expliquer clairement pourquoi c'est gratuit (programme pilote, pas charite).
5. **Le support reactif fait la difference** : Repondre en moins de 2h sur WhatsApp pendant le pilote.
6. **Parler leur langage** : Utiliser les termes du metier (lot, affinage, presure, lactosequum) et pas du jargon tech.
7. **Un pilote satisfait en ramene deux** : Le bouche a oreille est le meilleur canal d'acquisition en Algerie.
